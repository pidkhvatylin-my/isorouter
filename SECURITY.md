# Security Policy

## Supported Versions

Only the latest published `1.x` release of each `@isorouter/*` package
receives security fixes.

## Reporting a Vulnerability

Please **do not** open a public issue for security vulnerabilities.

Instead, report it privately via
[GitHub Security Advisories](https://github.com/pidkhvatylin-my/isorouter/security/advisories/new),
or email michel.podkhvatilin@gmail.com with details and reproduction steps.

We'll acknowledge your report as soon as possible and aim to ship a fix or
mitigation promptly, crediting you (unless you'd prefer otherwise) once it's
released.
